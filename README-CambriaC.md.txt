This repository contains 2 subfiles of Homeworks and Labs. Within the homework and lab folders, there are other folders that lists the specific homework and lab numbers. Within the specific homework and lab folders, there contains files that include code and figures that relate to that specific lab/homework. The code will either be in the form of text files or .py files. The main purpose of the repo is to store all files created for the APPM 4600 Numerical Analysis class.
